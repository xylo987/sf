无聊就写点小练习，锻炼脑子。

各种编程语言都学，都写，已经无所谓语言了，只想看看各种语言之间的差异。

最喜欢的编程语言喜欢程度从大到小：

VB.NET > C > Python > Javascript > Haskell

C最快，VB.NET借助.NET实现，第二快，VISUAL STUDIO宇宙无敌IDE，编写WinForm程序简直无敌，
Python，可以练习写设计模式，做一些快速的项目，VB.NET不能看代码写的好，而是为了出活。

Javascript也是。

Haskell可以用来思考边界问题，毕竟都是递归，才能干活，如果不考虑边界，递归就没有退出。

所以，综合起来，Python用来编写面向对象和设计模式相关的代码，C用来玩，VB用来写业务。

造轮子，用C，VB.NET，Python用来调用即可。Javascript用来处理浏览器与服务器的交互。
